## **$.udb.checkVersion()**

Checks the USoft version. If the version used on the browser differs from the version used on the server, an error message is displayed.

Returns the 'this' object.

*Syntax*

```js
.checkVersion()
```

*Example*

```js
$.udb.checkVersion();
```
