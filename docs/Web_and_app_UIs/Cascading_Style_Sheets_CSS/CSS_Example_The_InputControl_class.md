# CSS Example: The InputControl class

Here is an example of some style sheet rules that apply to the input field of the InputControl class:

```language-css
input.InputControl{
    background-color: white;
    color: black;
    border-width: 1px;
    border-color: black;
    padding-left: 5pt;
}
```
