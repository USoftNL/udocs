# Rename()




:::note

This article is about the **Rename** action in the [USoft Action Language](/docs/Task_flow/Action_Language_reference/USoft_Action_Language.md).

:::

## **Rename()**

Renames the current item in a tree view control.

*Syntax*

```
Rename()
```

*Exposed by*

- Tree view controls
