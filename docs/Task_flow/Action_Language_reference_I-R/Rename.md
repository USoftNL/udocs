# Rename()



> [!NOTE]
> This article is about the **Rename** action in the [USoft Action Language](/docs/Task%20flow/Action%20Language%20reference/USoft%20Action%20Language.md).

## **Rename()**

Renames the current item in a tree view control.

*Syntax*

```
Rename()
```

*Exposed by*

- Tree view controls
