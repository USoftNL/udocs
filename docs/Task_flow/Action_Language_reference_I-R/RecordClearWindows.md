# RecordClearWindows()



> [!NOTE]
> This article is about the **RecordClearWindows** action in the [USoft Action Language](/docs/Task_flow/Action_Language_reference/USoft_Action_Language.md).

## **RecordClearWindows()**

Clears data in all windows of the application.

*Syntax*

```
RecordClearWindows()
```
