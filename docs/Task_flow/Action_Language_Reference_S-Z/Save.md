# Save()



> [!NOTE]
> This article is about the **Save** action in the [USoft Action Language](/docs/Task%20flow/Action%20Language%20reference/USoft%20Action%20Language.md).

## **Save()**

Calls the Save action of the contained OLE object.

*Syntax*

```
Save()
```

*Exposed by*

- OLE containers
