# Accept()



> [!NOTE]
> This article is about the **Accept** action in the [USoft Action Language](/docs/Task%20flow/Action%20Language%20reference/USoft%20Action%20Language.md).

## **Accept()**

Executes a double-click on the current item.

*Syntax*

```
Accept()
```

*Exposed by*

- Tree views
- List views
