# Copy()



> [!NOTE]
> This article is about the **Copy** action in the [USoft Action Language](/docs/Task_flow/Action_Language_reference/USoft_Action_Language.md).

## **Copy()**

Calls the copy action of a contained OLE object.

*Syntax*

```
Copy()
```

*Exposed by*

- OLE containers
