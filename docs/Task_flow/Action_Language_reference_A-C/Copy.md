# Copy()



> [!NOTE]
> This article is about the **Copy** action in the [USoft Action Language](/docs/Task%20flow/Action%20Language%20reference/USoft%20Action%20Language.md).

## **Copy()**

Calls the copy action of a contained OLE object.

*Syntax*

```
Copy()
```

*Exposed by*

- OLE containers
