# ActionDialogCancel()




:::note

This article is about the **ActionDialogCancel** action in the [USoft Action Language](/Task_flow/Action_Language_reference/USoft_Action_Language.md).

:::

## **ActionDialogCancel()**

Closes a dialog without doing or undoing anything.

*Syntax*

```
ActionDialogCancel()
```

*Exposed by*

- Dialogs
