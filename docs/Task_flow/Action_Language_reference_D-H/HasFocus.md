# HasFocus()




:::note

This article is about the **HasFocus** action in the [USoft Action Language](/docs/Task_flow/Action_Language_reference/USoft_Action_Language.md).

:::

## **HasFocus()**

Calls the HasFocus action of an OLE object.

*Syntax*

```
HasFocus()
```

*Exposed by*

- OLE containers
