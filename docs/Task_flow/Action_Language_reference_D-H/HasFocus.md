# HasFocus()



> [!NOTE]
> This article is about the **HasFocus** action in the [USoft Action Language](/docs/Task%20flow/Action%20Language%20reference/USoft%20Action%20Language.md).

## **HasFocus()**

Calls the HasFocus action of an OLE object.

*Syntax*

```
HasFocus()
```

*Exposed by*

- OLE containers
