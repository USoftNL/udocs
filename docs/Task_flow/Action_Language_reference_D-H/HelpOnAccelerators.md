# HelpOnAccelerators()




:::note

This article is about the **HelpOnAccelerators** action in the [USoft Action Language](/Task_flow/Action_Language_reference/USoft_Action_Language.md).

## **HelpOnAccelerators()**

Displays help on accelerator keys for the active window.

*Syntax*

```
HelpOnAccelerators()
```
