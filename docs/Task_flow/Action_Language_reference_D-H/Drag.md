# Drag()




:::note

This article is about the **Drag** action in the [USoft Action Language](/Task_flow/Action_Language_reference/USoft_Action_Language.md).

:::

## **Drag()**

Calls the Drag action of a contained OLE object.

*Syntax*

```
Drag()
```

*Exposed by*

- OLE containers
