# Drag()



> [!NOTE]
> This article is about the **Drag** action in the [USoft Action Language](/docs/Task%20flow/Action%20Language%20reference/USoft%20Action%20Language.md).

## **Drag()**

Calls the Drag action of a contained OLE object.

*Syntax*

```
Drag()
```

*Exposed by*

- OLE containers
